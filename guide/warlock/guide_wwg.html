前言

傳統咒術追求單發高傷害， 但我對咒術的理解是連發性能>單發傷害， 並不是不注重傷害，是在秒7的前提下，再來追求傷害， 但造價會非常高，不過一旦跨過那個門檻，就會有非常強大的輸出能力。

連鎖電擊
用電擊魔法向1名目標給予損害，當受到電擊損害的目標附近有其他敵人時，連鎖電擊會自動找到目標並給予損害，當敵人數量是複數時會隨機攻擊。技能等級越高連鎖的電擊次數也會增加，當最後的目標附近沒有其他目標時，會再攻擊該目標3次。
傷害公式: 技能等級傷害 x ([Base Lv] / 100)%
每次攻擊會依照100 x (9 - 連鎖次數)% 增加傷害
初次攻擊會在對象5x5範圍內尋找連鎖目標，二次攻擊後會在對象7x7範圍內尋找目標
獨立延遲0秒
共通延遲3秒
  • [Lv1]: 風屬性魔法傷害600%，5次連鎖，固定詠唱1秒，變動詠唱3.5秒
  • [Lv2]: 風屬性魔法傷害700%，6次連鎖，固定詠唱1秒，變動詠唱4秒
  • [Lv3]: 風屬性魔法傷害800%，7次連鎖，固定詠唱1秒，變動詠唱4.5秒
  • [Lv4]: 風屬性魔法傷害900%，8次連鎖，固定詠唱1秒，變動詠唱5秒
  • [Lv5]: 風屬性魔法傷害1000%，9次連鎖，固定詠唱1秒，變動詠唱5.5秒
資料來源:官方、幻想廳

傷害公式: (技能等級傷害 + INT) x ([Base Lv] / 100)%

連鎖電擊的第一個特性是無獨立延遲但是有非常高的高達3秒的共通延遲，所以才需要把減延遲撐到96%以上這麼高，再來就是每次傷害都是獨立計算，所以很容易觸發一些效果離如爆走魔法之類的，但魔增就只支援第一下，不過秒七的技能是不會用到魔增的。
連鎖電擊還有個特別的地方就是無詠的時候完全不會舉手，頭上也不會有字，很像用眼神殺人，很有魔王的架勢XD，也因為這樣沒有卡動作延遲的問題。
連鎖電擊單看公式會誤以為是會越電越痛，但實際上效果是越電越癢，以185等技能5級來說，實際上是 1850+800 = 2650% -> 2550% -> 2450% .....這樣下去，每次的彈跳增傷都是以基本的傷害為準，不是上一次的傷害。
靈魂爆炸則是各方面要求都比較低，所以只要達到連鎖電擊的秒7門檻，魂爆就OK了。

素質

無狀態
慈悲術+百合花+菠色克藥水+攻速汁
配點基本上是跟著配裝走，175時代我是完全捨棄防禦的純攻擊取向配法，反正不打PVP，死了就爬起來就好，185開放後加上加速術改強，就多了很多點數出來可以點VIT和LUK
這個配法不是唯一，我的思路是喝菠色克藥水、攻速汁，在平時有主教能夠不吃料理無詠+193，自己單走時吃料理+天賜加速卷也能無詠+193。
我採取的無詠方案是素質無詠(INT/2 + DEX >= 265)，因為要使用暴徒領巾已經綁定ID120，而且若是採用裝備無詠雖然實戰上比較方便，可是要犧牲很多其他屬性和傷害加成。
AGI其實在現今狀況已經不用點這麼高，但我為了觸發戰死斗的加攻速效果才點到90。

俗稱戰死頭，我個人認為是PVP首選帽子，不過我是PVP小白所以請自行斟酌。這頂帽子給的東西都很不錯，加魔攻又加魂爆傷害，又能附魔魔精減固詠0.5秒，但他錯就錯在生在這個有席琳套裝的時代，既生瑜，何生亮...，而且還不能交易，很多人不敢插王卡上去。
