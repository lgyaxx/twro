介紹

咒術是一個門檻很高的職業，平民裝幾乎沒什麼輸出，尤其是現在講求持續輸出的年代，沒有一定裝備基本上很難當主力打王手，畢竟RO已經不是當年那種拚一發技能就秒殺怪物的遊戲了。


配點

風念咒術就是以連鎖電擊(風)和靈魂爆炸(念)為主力技能的咒術，這兩個技能是在充滿超高獨立延遲的咒術技能樹中，難得的無獨立延遲技能，換句話說就是可以秒7。
所以原則就是把連鎖電擊、靈魂爆炸、終極念力點滿，其他就看個人。不過在影子連鎖電擊套問世後，靈魂爆炸基本上只有電不到的時候才會拿出來用了。

技能
連鎖電擊

用電擊魔法向1名目標給予損害，當受到電擊損害的目標附近有其他敵人時，連鎖電擊會自動找到目標並給予損害，當敵人數量是複數時會隨機攻擊。
技能等級越高連鎖的電擊次數也會增加，當最後的目標附近沒有其他目標時，會再攻擊該目標3次。

傷害公式: 技能等級傷害 x ([Base Lv] / 100) + 100 x (9 - 連鎖次數)%
(每次攻擊會依照100 x (9 - 連鎖次數)% 增加傷害)

初次攻擊會在對象5x5範圍內尋找連鎖目標，二次攻擊後會在對象7x7範圍內尋找目標

獨立延遲0秒
共通延遲3秒
  • [Lv1]: 風屬性魔法傷害600%，5次連鎖，固定詠唱1秒，變動詠唱3.5秒
  • [Lv2]: 風屬性魔法傷害700%，6次連鎖，固定詠唱1秒，變動詠唱4秒
  • [Lv3]: 風屬性魔法傷害800%，7次連鎖，固定詠唱1秒，變動詠唱4.5秒
  • [Lv4]: 風屬性魔法傷害900%，8次連鎖，固定詠唱1秒，變動詠唱5秒
  • [Lv5]: 風屬性魔法傷害1000%，9次連鎖，固定詠唱1秒，變動詠唱5.5秒
資料來源:官方、幻想廳
靈魂爆炸

對遠距離的敵人利用念力給予2次損害，對象處於白色監獄效果時會受到2倍的損害。

傷害公式: (技能等級傷害 + INT) x ([Base Lv] / 100)%

對[白色監獄]狀態的目標造成2倍傷害
獨立延遲0秒
共通延遲0.5秒
  • [Lv1]: 念屬性魔法傷害500%，範圍3x3，固定詠唱0秒，變動詠唱2秒
  • [Lv2]: 念屬性魔法傷害600%，範圍3x3，固定詠唱0秒，變動詠唱2秒
  • [Lv3]: 念屬性魔法傷害700%，範圍3x3，固定詠唱0秒，變動詠唱2秒
  • [Lv4]: 念屬性魔法傷害800%，範圍5x5，固定詠唱0秒，變動詠唱2秒
  • [Lv5]: 念屬性魔法傷害900%，範圍5x5，固定詠唱0秒，變動詠唱2秒
資料來源:官方、幻想廳

連鎖電擊的第一個特性是無獨立延遲但是有非常高的高達3秒的共通延遲，所以才需要把減延遲撐到96%以上這麼高，再來就是每次傷害都是獨立計算，所以很容易觸發一些效果離如爆走魔法之類的，但魔增就只支援第一下，不過秒七的技能是不會用到魔增的。
連鎖電擊還有個特別的地方就是無詠的時候完全不會舉手，頭上也不會有字，很像用眼神殺人，很有魔王的架勢XD，也因為這樣沒有卡動作延遲的問題。
連鎖電擊單看公式會誤以為是會越電越痛，但實際上效果是越電越癢，以185等技能5級來說，實際上是 1850+900 = 2750% -> 2650% -> 2550% .....這樣下去，每次的彈跳增傷都是以基本的傷害為準，不是上一次的傷害。
靈魂爆炸則是各方面要求都比較低，所以只要達到連鎖電擊的秒7門檻，魂爆就OK了。


素質

為了達到每秒7次技能(秒7)，必須要無詠唱、延遲96%以上以及攻速(ASPD)193，配點就以這三個目標下去配，我V配108是為了觸發超越斗的5%延遲。
現在我為了穿影子職業2.0和超越套，已經無法堅持不使用料理了，目前固定會吃的就是ALL+10的終極料理，以及菠色克藥水和攻速濃縮汁。
